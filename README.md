# 🦊 renewal-blog
> Next.js + TypeScript + Tailwind + shadcn/ui 기반으로 제작된 포트폴리오 중심 블로그

## ✨ 주요 기능

```
🙋‍♀️ About - 소개 및 경력 정리
🎨 Portfolio - 실제 작업물 정리
📋 Project - 주요 프로젝트 목록과 설명
📬 Contact - 이메일, GitHub 연락처 정보
```


## 🌱 Stacks
- React
- Next.js (App Router)
- TypeScript
- Tailwind
- shadcn/ui + HeroUI
- Framer Motion
- Lenis
